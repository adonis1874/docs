## Searching images with Pinterest Lens

You can reverse search an image/member avatar via Pinterest Lens with the `pinsearch` command.

```
// Search an image
pinsearch [attachment or url]

// Search a member's avatar
pinsearch [member mention]
```

## Creating a Pinterest feed

bleed allows your server to recieve images from a user's Pinterest board to a Discord channel.
This can be used for things like auto profile picture publishing.

You can add a Pinterest feed to a channel with the `pinterest add [channel] [handle]` command.

### Customization Options

<AccordionGroup>
  <Accordion title="Embeds">
    This command is used to enable/disable embeds for new pins.

    ```
    pinterest embeds [channel] [yes or no]
    ```
  </Accordion>

  <Accordion title="Custom Messages">
    <Info>Using the `--pingable` flag allows you to mention roles!</Info>

    This command is used to set a custom message for new pins.

    You can view this message with `pinterest message view [handle]`

    ```
    pinterest message [handle] [message] [--pingable on/off]
    ```
  </Accordion>
</AccordionGroup>

## Related Commands

<AccordionGroup>
  <Accordion title="Remove a Pinterest feed">
    This command is used to remove a Pinterest feed from a channel.

    ```
    pinterest remove [channel] [handle]
    ```
  </Accordion>

  <Accordion title="List your server's Pinterest feeds">
    This command is used to list all Pinterest feeds in your server.

    ```
    pinterest list
    ```
  </Accordion>
</AccordionGroup>
