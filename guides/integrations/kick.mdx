## Creating a Kick feed

By adding a [Kick](https://kick.com) feed, bleed will notify members in a channel when a user starts streaming.

To add a feed from Kick, run `kick add #channel [username]`

### Customization Options

<AccordionGroup>
    <Accordion title="Custom Messages">
        Add a custom message that is sent to your channel alongside the notification.
        <Tip>Supports **raw text** and **embed code**! Add `--pingable on` to the command to ping a role in the message.</Tip>
        ```
        // Command syntax
        kick message [username] (message)

        // Raw text example without ping
        kick message [username] Bryce is now live!

        // Embed example with ping
        kick message [username] {embed}$v{description: Bryce is now live!}$v{message:@everyone} --pingable on
        ```
        You can view the message set with `kick message view [username]`
    </Accordion>
</AccordionGroup>

### Related Commands

<AccordionGroup>
    <Accordion title="Remove a Kick feed">
        Used to remove a Kick feed from a channel
        ```
        kick remove #channel [username]
        ```
    </Accordion>

    <Accordion title="List all Kick feeds">
        Used to list all Kick feeds within your server
        ```
        kick list
        ```
    </Accordion>
</AccordionGroup>
